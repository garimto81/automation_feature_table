7. Error Handling Summary

Error Type When Detection Response
Path Not Found Start button click Path.exists() st.error display, abort start
No Files Found File scan rglob result = 0 st.warning("No files found")
JSON Parse Error Scan / Simulation json.loads exception Skip file, log error
Write Failure Simulation OSError Retry 3 times (5s interval), then ERROR status
NAS Disconnection Simulation Retry failure ERROR status, log message
Memory Overflow Large file processing MemoryError Python exception, process termination
tkinter Unavailable App startup ImportError Hide folder browse buttons
Thread Conflict Start button click status check Button disabled (prevented)
